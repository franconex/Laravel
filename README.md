## Requisitos

- PHP 8.4+
- Composer
- Node.js & npm
- Base de datos (MySQL, PostgreSQL, SQLite, etc.)

## Instalación

```bash
composer install
npm install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
